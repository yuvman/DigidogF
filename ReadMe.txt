DigiDog App

Overview
The mobile application has been developed using React Native and Expo. It allows the user to play with Milo. User can select various toys. The app shows Milo's mood, which changes evry 15 seconds. The user can improve Milo's mood by tapping on a toy. Additionally, the app features a points system to track user progress.

Dependencies
React Native
Expo
expo-av
@react-navigation/native
Special File Paths
Dog GIFs: The GIFs representing Milo's different moods are located in the ./assets directory.
Dog Mood Sounds: The sound files corresponding to Milo's different moods are located in the ./assets directory.
Connectivity Requirements
The app does not have any specific connectivity requirements and can be used offline.

Installation
To run the app locally, follow these steps:

Clone the repository.
Navigate to the project directory in your terminal.
Install dependencies by running npm install Start the Expo development server by running expo start.
Scan the QR code using the Expo Go app on your mobile device or run the app in a simulator.
Usage
Once the app is running, users can:

View Milo's current mood and mood message.
Play with Milo by selecting toys from the toy grid.
See their points increase each time they play with a toy.
